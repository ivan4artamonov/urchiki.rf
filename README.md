# urchiki.rf

Laravel-проект сайта `urchiki.rf`.

## Что в репозитории

- Основной код приложения на Laravel.
- Конфигурация локальной разработки и миграции базы данных.
- Служебная папка `claude` для материалов, связанных с работой нейросети.

## Папка `claude`

В папке `claude` лежат файлы, сгенерированные этой нейросетью.
Там могут быть:

- контекст по задачам;
- технические задания (ТЗ);
- прототипы;
- другие материалы, которые спускаются из проекта в Claude, где формируется стратегия.

Папка `claude` добавлена в `.gitignore` и не индексируется Git.

## Локальный запуск

- Установить зависимости: `composer install` и `npm install`
- Настроить окружение: скопировать `.env.example` в `.env`
- Сгенерировать ключ приложения: `php artisan key:generate`
- Выполнить миграции: `php artisan migrate`
- Запустить сервер: `php artisan serve`
